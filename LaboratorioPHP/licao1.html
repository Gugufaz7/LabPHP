Lição 1 - Introdução ao PHP

PHP (Hypertext Preprocessor) é uma linguagem de script de código aberto amplamente utilizada, especialmente adequada para o desenvolvimento web e que pode ser embutida no HTML.

O que é PHP?

PHP é uma linguagem de programação do lado do servidor que é usada para criar páginas web dinâmicas e interativas. Foi originalmente criada por Rasmus Lerdorf em 1994 e desde então evoluiu significativamente.

Características do PHP

  • Fácil de aprender e usar
  • Ampla compatibilidade com servidores web e sistemas operacionais
  • Suporte a uma ampla gama de bancos de dados
  • Comunidade ativa e grande quantidade de documentação

Exemplo de Código PHP

O exemplo a seguir mostra um simples script PHP que exibe "Olá, Mundo!" na tela:

<?php
echo "Olá, Mundo!";
?>

Como Executar PHP

Para executar um script PHP, você precisa de um servidor web com suporte a PHP, como o Apache, e um navegador web. Você pode instalar um pacote como o XAMPP ou WAMP que inclui o Apache, PHP e MySQL.

Salve o código acima em um arquivo com a extensão .php e coloque-o no diretório raiz do seu servidor web. Acesse o arquivo através do navegador para ver o resultado.

Variáveis e Tipos de Dados

<?php
$nome = "Augusto";
$idade = 30;
$altura = 1.75;
$casado = true;

echo "Nome: $nome\n";
echo "Idade: $idade\n";
echo "Altura: $altura\n";
echo "Casado: " . ($casado ? 'Sim' : 'Não') . "\n";
?>

Estruturas de Controle

<?php
$nota = 85;

if ($nota >= 90) {
    echo "Aprovado com louvor!";
} elseif ($nota >= 60) {
    echo "Aprovado";
} else {
    echo "Reprovado";
}
?>

Conclusão

PHP é uma linguagem poderosa e flexível que pode ser usada para criar uma ampla variedade de aplicações web. Com sua sintaxe simples e ampla compatibilidade, é uma excelente escolha para desenvolvedores de todos os níveis.
